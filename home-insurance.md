# Home Insurance

Insurance that covers damages from fire, theft, floods, and natural disasters to buildings and household contents.

## Building Insurance

Coverage for the physical structure of the property.

**Required Information:**
- Owner Name
- ID Number
- Phone Number
- Email Address
- Property Address
- Building Type
- Construction Year
- Building Area (sq meters)
- Number of Floors
- Building Value
- Property Ownership Document (Image)
- Property Photos (Multiple Images)
- ID Copy (Image)

## Contents Insurance

Coverage for household items and personal belongings.

**Required Information:**
- Owner Name
- ID Number
- Phone Number
- Email Address
- Property Address
- Contents Value Estimate
- High-Value Items List
- Property Ownership/Rental Agreement (Image)
- Contents Photos (Multiple Images)
- ID Copy (Image)
