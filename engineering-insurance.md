# Engineering Insurance

Insurance that covers construction projects, equipment, and industrial machinery against damages during construction and installation.

## Contractors All Risks Insurance

Coverage for construction and civil works projects.

**Required Information:**
- Company Name
- Company Registration Number
- Contact Person Name
- Phone Number
- Email Address
- Project Name
- Project Location
- Project Value
- Contract Value
- Project Start Date
- Project Duration
- Project Type
- Contract Agreement (Image)
- Project Plans (Images)
- Trade License (Image)
- Company Registration (Image)

## Machinery Breakdown Insurance

Coverage for industrial machinery and equipment.

**Required Information:**
- Company Name
- Company Registration Number
- Contact Person Name
- Phone Number
- Email Address
- Machinery Type
- Machinery Make and Model
- Year of Manufacturing
- Machinery Value
- Installation Location
- Usage Purpose
- Machinery Invoice (Image)
- Machinery Photos (Multiple Images)
- Maintenance Records (Image)
- Trade License (Image)
- Company Registration (Image)
