# Marine Insurance

Insurance that covers goods and cargo transported by sea and ships against maritime transport damages.

## Cargo Insurance

Coverage for goods transported by sea.

**Required Information:**
- Company Name
- Company Registration Number
- Contact Person Name
- Phone Number
- Email Address
- Cargo Type
- Cargo Value
- Origin Port
- Destination Port
- Shipping Date
- Vessel Name
- Invoice (Image)
- Packing List (Image)
- Bill of Lading (Image)
- Trade License (Image)

## Hull and Machinery Insurance

Coverage for ships and vessels.

**Required Information:**
- Owner/Company Name
- Company Registration Number
- Contact Person Name
- Phone Number
- Email Address
- Vessel Name
- Vessel Type
- Year Built
- Vessel Value
- Engine Details
- Usage Purpose
- Vessel Registration (Image)
- Vessel Survey Report (Image)
- Vessel Photos (Multiple Images)
- Ownership Document (Image)
