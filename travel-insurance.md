# Travel Insurance

Insurance that provides coverage for accidents, illness, lost luggage, and trip cancellation for domestic and international travel.

## Single Trip Insurance

Coverage for one-time travel.

**Required Information:**
- Customer Name
- ID Number
- Passport Number
- Date of Birth
- Phone Number
- Email Address
- Destination Country
- Travel Start Date
- Travel End Date
- Trip Purpose
- ID Copy (Image)
- Passport Copy (Image)

## Annual Multi-Trip Insurance

Coverage for multiple trips within one year.

**Required Information:**
- Customer Name
- ID Number
- Passport Number
- Date of Birth
- Phone Number
- Email Address
- Coverage Start Date
- Frequent Destinations
- ID Copy (Image)
- Passport Copy (Image)
